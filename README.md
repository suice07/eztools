# eztools
easy tools for everyday use

## Change and Build
```
python setup.py build
python setup.py sdist
```
## Installation
build and install step by step
```
cd eztools/dist
pip install eztools-0.1.1.tar.gz
```
or you can use the script
```
bash install.sh
```
